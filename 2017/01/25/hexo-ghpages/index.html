搭建博客(Hexo + github pages + 域名绑定)

Jan 25, 2017 | learn | Hits

之前购买了一个域名bebee.cc一直未用，今天折腾了一下，把Hexo博客挂上github pages，免费的值得折腾。

具体流程如下：

  • 搭建hexo博客，这里不介绍了，根据Hexo官网一步步搭建，再上网选择自己喜欢的style主题，就可以了。
  • 创建Github Pages 免费的静态站点，其特点：免费-免费-免费。创建 Github Pages 比较简单，注册github账号再创建一个yourusername.github.io仓库就行了。
  • 域名绑定，github pages 必须要有 CNAME 文件。在hexo博客本地站点目录里 source 目录下添加一个 CNAME 文件，不带后缀，里面添加你的域名信息（不加http://), 比如写 bebee.cc 就可以了，然后hexo d 到github.
    下一步注册 DNSpod ，然后添加域名，添加记录即可。按照如下图条目添加就可以了(注意yourusername.github.io的yourusername是你自己的github账号名)。然后到浏览器输入自己的域名，如何还不能访问，等10分钟后再试试。
