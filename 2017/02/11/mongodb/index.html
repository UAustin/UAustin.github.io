Mongodb mac安装与连接

Feb 11, 2017 | Mongodb | Hits

用 Homebrew 进行安装
1、终端中运行命令进行安装：brew install mongodb
2、创建 /data/db 目录：sudo mkdir -p /data/db
3、设置 /data/db 目录权限： sudo chown -R dailin /data
4、启动MongoDB：mongod
从启动信息中可以看到MongoDB的默认端口是 27017
在浏览器中访问 http://localhost:27017/ 可以看到信息
5、连接Mongo: mongo
从第一个终端窗口中可以看到已经有一个客户端连接上了MongoDB
运行 which mongo
可以看到 mongo 已经添加到 $PATH 中了， mongo的相关命令在 /usr/local/bin 目录下。
